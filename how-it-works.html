How It Works

  1. Select a Page: Use the sidebar to choose a specific category like Study or Gym.
  2. Add Tasks: Click "+ Add Plan" for any day and type your time and task.
  3. Print: Click the "Print Schedule" button to save or print a hard copy.
  4. Refresh: Refreshing the browser clears the board for a new planning session.
